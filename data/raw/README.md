# Coloque seus datasets aqui

## Instruções:

1. Baixe o dataset escolhido (Breast Cancer, PCOS, etc.)
2. Coloque o arquivo CSV nesta pasta
3. Renomeie se necessário para facilitar o código

## Exemplo:
- `breast_cancer.csv`
- `pcos_dataset.csv`
- `domestic_violence.csv`

## ⚠️ IMPORTANTE:
Não faça commit de datasets muito grandes (>100MB).
Use Git LFS ou forneça link para download no README.
